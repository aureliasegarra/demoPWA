## ⚙️ How to use 
### Project setup
```bash
npm install
```

### Compiles for development
```bash
npm run start
```
*Meeting on http://localhost:3000/*

### Launch the jsonServer 
```bash
npm run jsonserver
```
*Meeting on http://localhost:3001/technos*
